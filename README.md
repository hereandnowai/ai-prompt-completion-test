# HERE AND NOW AI Server Monitoring Dashboard

**AI is Good**

A modern, responsive server monitoring dashboard built with vanilla HTML, CSS, and JavaScript for live infrastructure observability. The dashboard helps operators review uptime, CPU load, memory use, disk pressure, network traffic, running processes, alerts, and application logs from one clean interface.

## Overview

This project demonstrates a polished monitoring experience for server health analytics, incident response, infrastructure status tracking, and AI-ready operations workflows. It is designed as a lightweight, dependency-free web dashboard that opens directly in a browser.

## Features

- Live server health cards for CPU, memory, disk, and network utilization.
- Canvas-based resource trend chart for monitoring recent system behavior.
- Service, storage, network, and maintenance status panels.
- Searchable process table with a simulated stop action.
- Alert queue with severity filtering, resolution, and silence controls.
- Application log viewer with level filtering and clear action.
- Monitoring settings modal for sample rate and notification preferences.
- HERE AND NOW AI branding with official colors, logo, favicon, slogan, and contact links.
